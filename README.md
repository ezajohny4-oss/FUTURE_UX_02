# FUTURE_UX_02
Mobile UI/UX booking app for an interior design studio, designed in Figma with onboarding, service listing, slot booking, and appointment confirmation flows.
##  Objective

The goal of this project is to engineer a highly intuitive, mobile-first appointment booking platform that completely eliminates booking friction for first-time users. Built with clean typography, clear informational hierarchy, and production-ready design principles, this system transforms a standard booking journey into a trustworthy, professional extension of an enterprise interior design agency.

## Real-World Scenario

* **Client:** `Aura & Arch Interior Design Studio`
* **Target Users:** Homeowners, commercial venue operators, and property managers looking to book specialized interior architectural consultations, space renovations, or office layout planning sessions.
* **Core Problem:** Standard inquiry forms cause user drop-off due to high friction, visual clutter, and ambiguous pricing structures.
* **The Solution:** A rapid, 4-stage mobile onboarding and confirmation funnel designed to maximize user conversion through contextual clarity.

---

## User Journey & Key Screen Architecture

### 1. Onboarding Screen (`01_Onboarding`)

* **UX Strategy:** Establishes instantaneous brand authority through high-quality, aspirational architectural imagery paired with clean typography.
* **Friction Reduction:** Minimizes structural onboarding drop-off by implementing frictionless one-tap **"Continue with Google"** and **"Continue with Apple"** Auto Layout button capsules.

### 2. Service Selection Screen (`02_Services_List`)

* **UX Strategy:** Features an accessible, top-anchored search matrix followed by a clean vertical scroll feed of unified service components.
* **Production Principles:** Every service option uses fixed-width structured cards containing concrete micro-copy data clusters—clearly outlining durations (`60 mins`), rates (`₹2,500/hr`), and crisp descriptions to remove conversion doubt.

### 3. Date & Time Slot Booking (`03_Slot_Booking`)

* **UX Strategy:** Organizes complex scheduling criteria into digestible, touch-friendly UI interactive spaces.
* **Accessibility Standards:** * Features an expansive horizontal calendar lane to toggle consecutive daily dates effortlessly.
* Groups time allocations into explicit, high-contrast structural pills partitioned by daily phases (*Morning*, *Afternoon*, *Evening*).
* Color-coded states distinctly communicate availability (Green stroke), selected choice (Solid forest green fill), and locked/past intervals (Subtle neutral grey fill).



### 4. Appointment Confirmation (`Appointment Confirmation`)

* **UX Strategy:** Reduces cognitive anxiety post-booking through immediate visual validation (bold green check success indicator).
* **Information Architecture:** Condenses vital appointment criteria (*Service specifications*, *Assigned Specialist*, *Scheduled Date*, and *Virtual Meeting coordinates*) into a secure, elevated container card, finalized by a high-contrast call-to-action button stack.

---

## Production Standards & Design System

* **Core Tool:** Figma
* **Typography Hierarchy:** Engineered utilizing **Inter** for clean, legible user interfaces and structural text lines, alongside **Playfair Display** to communicate sophisticated luxury across primary headlines.
* **Layout Mechanics:** Built entirely on **Figma Auto Layout**, utilizing responsive constraints, fixed-dimensional padding frameworks, and precise corner radii (`16px` cards, `27px`/`30px` pill capsules) .
